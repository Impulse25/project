# Модуль «Аналитика и отчётность»

Структура модуля:

- `index.php` — основная страница модуля и вывод интерфейса;
- `includes/helpers.php` — вспомогательные функции, проверка таблиц/полей, расчёт периодов и критериев;
- `assets/css/analytics.css` — оформление модуля, светлая/тёмная тема, печать;
- `assets/js/analytics.js` — переключение темы и экспорт таблицы в Excel.

Модуль подключается к общей базе портала через `../config/db.php`, поэтому отдельный файл подключения внутри папки `analytics` не дублируется.
